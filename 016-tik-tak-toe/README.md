# ⭕ Tic-Tac-Toe

🔗 **Live Demo:** https://sarfraz-016-tik-tak-toe.netlify.app/

A classic Tic-Tac-Toe game built with pure HTML, CSS, and JavaScript. Play against a friend on the same device with a stylish neon-themed interface and a color-changing accent system.

## Features
- Two-player turn-based gameplay (O goes first)
- Win detection with winner announcement overlay
- Neon glow effects and custom handwritten font
- Color theme switcher — click the circle button to cycle through neon colors
- Reset game anytime with the Reset button
- New Game button appears after a winner is found

## Tech Stack
HTML · CSS · Vanilla JavaScript

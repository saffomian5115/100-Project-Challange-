# 🖼️ Photo Gallery

A photo gallery web app built with React, Tailwind CSS, and Node.js. It allows users to upload, view, sort, and delete images, with auto-generated thumbnails for faster loading. The project is structured with a frontend and backend, using Axios for communication between the two.

## Features
- Multiple image upload
- Auto-generated thumbnails (via Sharp)
- Sort gallery by name, size, upload time, or type
- Image preview and delete functionality
- Responsive grid layout

## Tech Stack
**Frontend:** React · Tailwind CSS · Axios · React Hot Toast · Lucide Icons
**Backend:** Node.js · Express · MySQL2 · Multer · Sharp

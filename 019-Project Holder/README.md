# 🗂️ Project Holder

A React-based hub app that collects and showcases multiple mini-projects in one place. Features user authentication, dark/light theme toggle, and a dashboard to browse and launch each mini-project.

## Features
- Dashboard with 6 mini-projects (Payment Method, Counter, Notice Board, Testing, Counter with useEffect, API Fetch)
- User authentication mock (login/logout) with protected routes
- Dark/Light theme toggle (persisted to localStorage)
- Built with React Router for seamless navigation

## Mini Projects Included
1. **Payment Method** — UI for selecting payment options
2. **Counter App** — Simple increment/decrement counter
3. **Notice Board** — Displays notices from a predefined list
4. **Testing** — A testing playground component
5. **Counter with useEffect** — Counter using the useEffect hook
6. **API Fetch Random Posts** — Fetches and displays random post data from an API

## Tech Stack
React · React Router · Tailwind CSS · Vite

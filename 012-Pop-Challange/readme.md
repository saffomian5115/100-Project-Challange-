# 🎈 Balloons Pop Challenge

🔗 **Live Demo:** https://sarfraz-012-pop-challange.netlify.app/

A fun balloon-popping browser game. Pop balloons before time runs out to score points — every pop spawns two new balloons to keep the challenge going!

## Features
- Selectable game duration (15s / 30s / 1 min / 2 min)
- Live score and countdown timer with sound effects
- Pop animation
- High score saved per time mode (localStorage)
- Responsive design

## Tech Stack
HTML · CSS · Vanilla JavaScript

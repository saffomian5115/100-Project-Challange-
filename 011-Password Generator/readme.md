# 🔐 Password Generator

🔗 **Live Demo:** https://sarfraz-011-password-generator.netlify.app/

A simple and customizable password generator built with React and Tailwind CSS. Adjust the password length and choose whether to include numbers and special characters, then copy the generated password with a single click.

## Features
- Adjustable password length (1–100 characters)
- Toggle numbers on/off
- Toggle special characters on/off
- One-click copy to clipboard
- Clean, responsive UI

## Tech Stack
React · Tailwind CSS · Vite

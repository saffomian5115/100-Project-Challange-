# Password Strength Checker

Live Demo: https://sarfraz-007-password-strength-checker.netlify.app/

A simple tool that checks how strong a password is based on length, uppercase/lowercase letters, numbers, and special characters. Shows a strength bar and label in real time.

## Features
- Real-time password strength checking
- Strength bar with color and label (weak, normal, strong, good)
- Show/hide password toggle

## Tech Used
- HTML
- CSS
- JavaScript

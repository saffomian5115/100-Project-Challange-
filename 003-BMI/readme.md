# BMI Calculator

Live Demo: https://sarfraz-003-bmi-calculator.netlify.app/

A simple BMI (Body Mass Index) calculator. Enter your height and weight to instantly see your BMI value and category (Underweight, Normal, Overweight, Obese).

## Features
- Calculate BMI from height (cm) and weight (kg)
- Shows BMI category
- Input validation for invalid values

## Tech Used
- HTML
- CSS
- JavaScript

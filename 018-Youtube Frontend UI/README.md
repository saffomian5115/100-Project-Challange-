# ▶️ YouTube UI Clone

🔗 **Live Demo:** https://sarfraz-018-youtube-ui-clone.netlify.app/

A front-end clone of the YouTube homepage interface built with pure HTML and CSS. Recreates the look and feel of YouTube's layout including the navbar, sidebar, category chips, and video grid.

## Features
- YouTube-style top navbar with search bar, mic, create, bell, and profile icons
- Collapsed sidebar with Home, Shorts, Subscriptions, and You navigation
- Category chip bar (All, Music, Live, etc.)
- Responsive video grid with thumbnails, channel icons, titles, and view counts
- Hover effects on interactive elements

## Tech Used
- HTML
- CSS

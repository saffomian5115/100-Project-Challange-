# Quiz App

Live Demo: https://sarfraz-005-quiz.netlify.app/

A fun quiz app where you choose the number of questions, category, and difficulty level, then test your knowledge with a timer for each question.

## Features
- Choose number of questions, category, and difficulty
- Timer for each question
- Score tracking at the end
- Sound effects

## Tech Used
- HTML
- CSS
- JavaScript
- Open Trivia DB API

# Random Jokes

Live Demo: https://sarfraz-006-random-joke.netlify.app/

A fun web app that fetches and displays random jokes from an API. Includes sound effects for good and bad reactions, with an animated background.

## Features
- Fetches random jokes from JokeAPI
- Good/bad reaction buttons with sound effects
- Animated floating balls background
- Loading animation while fetching

## Tech Used
- HTML
- CSS
- JavaScript
- JokeAPI

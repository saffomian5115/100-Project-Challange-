# Color Changer

Live Demo: https://sarfraz-009-color-changer.netlify.app/

A simple React app that changes the background color of the page when you click on different color buttons. Also has a random color button for surprise colors.

## Features
- Click buttons to change background color
- Random color generator button
- Built with React and styled using Tailwind CSS

## Tech Used
- React
- Tailwind CSS
- Vite

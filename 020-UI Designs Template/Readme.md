# UI Design Gallery

Live Demo: https://sarfraz-20-ui-design-template.netlify.app/

A gallery showcasing 14 different UI design styles (Material, Glassmorphism, Neubrutalism, Neumorphism, Cyberpunk, Vaporwave, and more), each with its own live demo page and downloadable design spec.

## Features
- 14 different UI design style demos
- Material Design styled gallery with cards
- Preview and download design spec (.md) for each style
- Random style picker button

## Tech Used
- HTML
- CSS
- JavaScript

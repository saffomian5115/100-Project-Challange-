# 🎯 Guess the Number

🔗 **Live Demo:** https://sarfraz-017-guess-the-number.netlify.app/

A fun number guessing game built with pure HTML, CSS, and JavaScript. Try to guess the random number between 0 and 1000 with progressive hints that tell you how close you are!

## Features
- Random number between 0 and 1000 every round
- Progressive hint system — hints get closer as you near the answer
- Keyboard support (Enter to submit, auto-focus on number keys)
- Celebratory win screen with the correct answer
- New Game button to play again
- Input validation for out-of-range numbers

## Tech Stack
HTML · CSS · Vanilla JavaScript
